‚úÖ Core Functionality
Add Tasks: Easily add new tasks using the input field and "Add Task" button (or by pressing Enter).

Edit Tasks: Tasks can be modified directly on the list:

Click the "Edit" button to make the task text editable.

The button changes to "Save" during the editing process.

Mark as Complete: Click directly on the task text to toggle its completion status.

Completed tasks display a line-through and a muted color for visual separation.

Note: Editing an incomplete task automatically removes the "completed" status.

Delete Tasks: Remove tasks permanently using the dedicated "Delete" button next to each item.
